# Etsy vs eBay Strategy

## Platform Differences

| Feature | eBay | Etsy |
|---------|------|------|
| **Audience** | Bargain hunters, collectors | Art buyers, home decorators |
| **SEO** | Item specifics, titles | Tags, titles, attributes |
| **Pricing** | Competitive, auction option | Fixed price, premium acceptable |
| **Best for** | Vintage items, frames | Art prints, handmade |
| **Fees** | ~13% | ~6.5% + listing fees |

## Current Setup

### eBay (Curious-Things)
- **Listings:** ~1700 total
- **Art prints:** 150 in "Giclee Art Prints" category
- **Variants:** 5 sizes each (4x6, 5x7, 8x10, 11x14, 16x20)
- **Strategy:** Volume, competitive pricing

### Etsy (CuriousThings2023)
- **Listings:** ~50 art prints migrated
- **Goal:** Move remaining 100 from eBay
- **Challenge:** Vendoo doesn't handle variants well
- **Strategy:** Curated sets, premium positioning

## Migration Workflow (Current)

1. Move eBay listing to Vendoo
2. Edit for Etsy (remove eBay coupon codes)
3. Upload as draft
4. Manually add variants in Etsy
5. Publish

**Pain points:**
- Manual variant creation
- Can't bulk edit
- Time consuming

## Optimization Opportunities

### Etsy-Specific
- Use all 13 tags
- Fill all attributes (style, room, occasion)
- Styled lifestyle photos (not just product on white)
- Bundle sets sell better than singles

### eBay-Specific
- Item specifics matter for search
- Promoted listings can help
- Auction + Buy It Now combo

## Bundle Strategy

Instead of single prints, create curated sets:

**"Dark Academia Gallery Wall Set - 5 Prints"**
- Higher perceived value
- Higher price point ($45-65 vs $15-20)
- Differentiates from competition
- Easier to manage (one listing, 5 sizes)

## Size Strategy

**Current:** 5 sizes per print
**Recommended:** Focus on 2-3 most popular

| Size | Use Case | Popularity |
|------|----------|------------|
| 8x10 | Standard frame size | High |
| 11x14 | Gallery walls | High |
| 16x20 | Statement piece | Medium |
| 5x7 | Desk/small spaces | Low |
| 4x6 | Postcards/gifts | Low |
