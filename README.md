# Clawbot Knowledge Base

Shared knowledge repository for Alex (hosted) and local Mac Mini agent.

## Folder Structure

- **security/** - Hardening guides, port configs, firewall rules
- **sourcing/** - Auction tracking, frame sourcing, pricing guides
- **content/** - Social media ideas, content calendar, trends
- **etsy-ebay/** - Listing optimization, SEO, platform-specific guides
- **public-domain/** - Art sources, search terms, curated collections
- **procedures/** - Step-by-step workflows, troubleshooting

## How to Use

1. Local agent should `git pull` periodically to stay updated
2. Both agents can reference these files for consistent knowledge
3. Adam can read/edit directly on GitHub
